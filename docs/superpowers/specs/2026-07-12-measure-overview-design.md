# Measure 概览页精简设计

## 目标

移除 Measure「概览与初始化」页中不会启动测量的运行示例，避免用户在地图中看不到可执行行为而误以为功能失效。

## 页面职责

- 概览页改名为「概览」，仅说明 `earth.useMeasure()` 获取并缓存测量工具实例的方式、会话结束方式与清理入口。
- 距离页负责三种距离测量方式的可运行示例。
- 面积页负责面积测量的可运行示例。
- 移除页负责 `clear()` 清空交互和结果的可运行示例。

## 变更范围

1. 删除只调用 `earth.useMeasure()`、没有测量操作的 `MeasureDemo.vue`。
2. 移除概览页的示例区块、示例锚点及相关导入。
3. 将导航、路由布局标题与相关测试中的「概览与初始化」统一为「概览」。
4. 保留概览页的构造器、类型及 `clear()` API 说明；不重复距离和面积的运行示例。

## 验证

- 更新文档结构测试中受影响的标题和示例断言。
- 执行相关 Vitest 测试与文档站构建，确认页面可构建且导航、锚点保持有效。
